EduTimeAI – Smart Classroom and Timetable Scheduler

Overview:
EduTimeAI is an AI-powered Smart Classroom and Timetable Scheduler that automates timetable generation for schools and colleges.
It ensures clash-free scheduling, considers subject difficulty, and balances teacher workload to improve learning efficiency.

Key Features:

AI-assisted timetable generation

Subject difficulty-based scheduling

Room booking and announcements system

Simple, user-friendly interface

Excel input/output support

Privacy and role-based access (Admin/Faculty/Student)

Tech Stack:

Frontend: HTML, CSS, JavaScript

Backend: Python (Flask)

Database: SQLite

Libraries: Pandas, Flask, OpenPyXL

Version Control: Git + GitHub
